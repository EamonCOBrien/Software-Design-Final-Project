What you need

If you've gotten this far without downloading the program from github, Congrats! We don't know how you got this far, but you should really go to github now, read the ReadMe and follow the instructions.


Now that you have the CVPaint downloaded you can start having fun. To get the most out of your CVPaint experience, you will need a magic wand. CVPaint's wand tracking uses color tracking, so as long as you have something with 2 different colors for the program to track (and you don't happen to be wearing one of those colors) you'll be fine. Your wand will only be limited by your imagination, but here are a few ideas to start you off:

  1. Two sticky notes back to back. You can use a pencil as a handle.
  2. A ping pong paddle
  3. Your phone (in a pinch)
  4. Magic Wand*

    5. *Recommended if you have acess to a 3D printer go to STL download here
