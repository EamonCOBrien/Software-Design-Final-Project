The Project

CVPaint is an interactive drawing program that allows users to draw on a camera feed using colored objects. It uses OpenCV for color tracking and drawing, and flask to run in a browser. We have been gradually implementing more advanced drawing features over the course of the project, including support for multiple line weights, color selection, and geometric shapes. Designing the buttons for the interface allowed us to explore classes and inheritance, and tracking the cursor let us use basic computer vision skills. Have fun and be creative

The Class

SoftDes, or Software design, is an introductory Engineering and computing class that focuses on the design of dynamic code. Students learn how to work with biology data, computative art, video games, and more throughout the semester with a combination of projects, toolboxes, labs, and other assignments. This project is the culmination of what we learned in SoftDes.
