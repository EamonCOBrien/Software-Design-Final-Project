The Project

CVPaint is an interactive drawing program that allows users to "draw" on their screen using colored "wands". This is a fun program meant to engage and entertain users by letting draw whatever their inner kid wants. Our program uses openCV, flask, and html to allow users to create interactive art with nothing but a computer and a "wand" (aka anything of color). We made our program accessible by encapsulating it in a web app and including multiple wand options. Including a downloadable STL file that user's can 3D print, how to make a sticky note wand, and other possible ideas. CVPaint is our team's final project and includes many of the techniques we learned throughout the semester.

The Class

SoftDes, or Software design, is an introductory Engineering and computing class that focuses on the design of dynamic code. Students learn how to work with biology data, computative art, video games, and more throughout the semester with a combination of projects, toolboxes, labs, and other assignments. This project is the culmination of what we learned in SoftDes.
